# Fund Your Goals

Goal-based investment planning platform for families.

**Live:** [fundyourgoals.in](https://fundyourgoals.in)

## Tech Stack
- Next.js 14 + React 18
- Vercel (hosting)
- Notion API (blog CMS)
- Web3Forms (email submissions)
- Google OAuth (admin auth)
